### SOCK - 모바일 암호화폐 지갑 앱

---

## ✍️ 프로젝트 소개
SOCK은 숙명여대와 HiFiveLab이 참여한 산학 협력 프로젝트로 암호화폐 초보자도 쉽게 이용할 수 있는 모바일 암호화폐 지갑 앱입니다.


## 🕰️ 개발 기간
2025.4.1 ~ 2025.6.30


## 📌 주요 기능

1. **비밀번호 설정 및 입력**

   
2. **지갑 생성**

  
3. **복구**


4. **토큰 보내기**

   
5. **받기**
   

6. **도움말**
